### 安装Python

如果您使用ubuntu系统

```
sudu apt-get python3
```

如果您使用centos系统

```
sudo yum -y install python3
```

如果您使用的是linux桌面发行版

[Download Python | Python.org](https://www.python.org/downloads/)

如果您使用macos系统

brew安装

```
brew install python3
```

pkg安装

[Download Python | Python.org](https://www.python.org/downloads/)

如果您使用的是Windows系统

[Download Python | Python.org](https://www.python.org/downloads/)
